## flex-Basis

flexBasis是一种在主轴上提供项目默认大小的与轴无关方法。如果父容器的flexDirection为row，则设置子项的flexBasis类似于设置子项的宽度；如果父容器的flexDirection为column，则设置子项的flexBasis类似于设置子项的高度。项目的flexBasis是在执行任何flexGrow和flexShrink计算之前，项目的默认大小，即在尺寸扩展和收缩计算之前的项目尺寸。
